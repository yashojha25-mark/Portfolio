# 🌐 Personal Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, projects, achievements, and learning journey as a developer.

## 🚀 About the Project

This portfolio was built to create a professional online presence and demonstrate my web development skills. It includes information about me, my technical skills, featured projects, and ways to connect with me.

The website is designed to be responsive, user-friendly, and visually appealing across all devices.

---

## ✨ Features

* Responsive Design for Mobile, Tablet, and Desktop
* Professional Home Page
* About Me Section
* Skills Showcase
* Projects Portfolio
* Contact Section
* Clean and Modern UI
* Easy Navigation

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yashojha25-mark/Portfolio.git
```

### Navigate to the Project

```bash
cd Portfolio
```

### Open in VS Code

```bash
code .
```

### Run the Project

Open `index.html` in your browser or use the Live Server extension in VS Code.
